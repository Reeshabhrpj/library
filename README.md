# library
readBOOK - An Online Library Management Application (using C# and ASP.NET)


Features :

1. Admin login
2. User login
3.sign up
4.logout
5.view Books
6.Author Management
7.Publisher management
8.Book Inventory
9.Delete Users (Only for the Admin)
10.Book Issue/Return (For Admin )
